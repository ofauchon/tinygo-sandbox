# WS2812 LEDs with SPI and DMA (STM32)

## Context

This TinyGo applications uses SPI Bus and DMA to drive WS2812 LED stripe.


## Support

* The DMA configuration is STM32 specific. 
* The code was tested on NUCLEO-WL55JC (STM32WL55)
